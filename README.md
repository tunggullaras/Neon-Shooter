# 🚀 Neon Assault

A fast-paced, neon-styled top-down shooter built entirely in vanilla HTML, CSS, and JavaScript — no dependencies, no build tools, just open and play.

![Game Screenshot](screenshot.png)

---

## 🎮 Gameplay

Survive endless waves of enemy ships. Each wave gets harder — more enemies, faster spawns, and tougher tanks. Dodge bullets, collect health pickups, and rack up the highest score you can.

---

## ✨ Features

- **3 enemy types** — Basic, Shooter, and Tank, each with unique behavior
- **Wave system** — Enemies scale in difficulty every wave
- **Dash mechanic** — Invincibility frames while dashing through bullets
- **Triple shot** — Unlocks automatically after reaching 500 score
- **Health pickups** — Random drops from defeated enemies
- **Particle effects** — Explosions, bullet trails, and dash effects
- **Neon HUD** — Live score, wave counter, health hearts, and dash cooldown bar

---

## 🕹️ Controls

| Input | Action |
|-------|--------|
| `W A S D` / Arrow Keys | Move |
| Mouse | Aim |
| Left Click (hold) | Shoot |
| `Space` / `Shift` | Dash (invincible!) |

---

## 👾 Enemy Types

| Enemy | Color | Behavior | Points |
|-------|-------|----------|--------|
| Basic | 🟠 Orange | Charges directly at player | 10 |
| Shooter | 🟣 Purple | Keeps distance, fires projectiles | 20 |
| Tank | 🔴 Red | Slow, takes 6 hits to kill | 30 |

## 🚀 Getting Started

No installation or server required.

```bash
# Clone the repo
git clone https://github.com/yourusername/neon-assault.git

# Open the game
cd neon-assault
open neon-assault.html
```

Or simply [download the HTML file](neon-assault.html) and open it in any modern browser.

## 🛠️ Built With

- **HTML5 Canvas** — Rendering
- **Vanilla JavaScript** — Game logic, physics, collision detection
- **CSS** — UI overlays and menu screens

No frameworks. No dependencies. Single file.
